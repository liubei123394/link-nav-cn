# vue-music-netease

基于 Vue 3 + Vite 的网易云音乐 PC 端首页静态还原项目。

## 技术栈

- Vue 3
- Vite
- Element Plus
- Axios

## 功能说明

- 顶部导航栏、搜索框、用户登录入口布局
- 首页轮播 Banner 自动切换
- 推荐歌单分区卡片展示、鼠标悬浮效果
- 基础样式适配电脑端响应式布局

## 运行方式

1. 安装依赖

```bash
cd vue-music-netease
npm install
```

2. 启动开发服务器

```bash
npm run dev
```

3. 访问页面

打开浏览器并访问 `http://localhost:4173`
